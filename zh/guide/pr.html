# 贡献指南

# 如何贡献代码

首先在项目主页 fork 本仓库，然后进行你的开发，请确保你的开发是在 dev 分支进行。本地测试无误后方可 pull request，我在检查通过后会合并代码到原始仓库。

# 要求

本项目代码需要符合vue.js 风格指南中优先级A和优先级B的要求以及本项目中的 ESlint 校验

TIP

由于项目初始没有仔细遵循风格指南，可能项目代码中已经存在不符合规范的代码，欢迎批评指正或者直接pr
